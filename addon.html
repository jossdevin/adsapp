install minecraft mods manually :
1. go to settings in minecraft
2. select the profile menu
3. file storage location and select external
4. Extract the mod addon file that has been downloaded
5. then look for the games and com.mojang folders after that, paste the files that were extracted earlier
6. Paste the behavior file into the behavior|_pack folder and the texture file into the resource_pack folder
7. After that, reopen Minecraft PE and the addon mod can be played
